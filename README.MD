# PassThruLogger

## Инструкция к применению:
1. распаковать в любую папку архив из [релиза](https://github.com/kirya-dev/J2534-PassThru-Logger/releases);
2. запустить PassThruLoggerControl.exe (единожды от имени админа для саморегистрации в реестре, о чем вежливо попросит);
3. выбрать в окне этой программы ваше устройство J2534;
4. запустить AsynchronousClient.exe;
5. из программы диагностики выбрать PassThruLogger;
6. запустить желаемую процедуру диагностики и смотреть логи в PassThruLoggerControl.exe.
